Apa itu adaptive dan responsive web design? Apakah kamu setuju bahwa teknik tersebut bagus? Mengapa atau mengapa tidak?

Adaptive web design berarti kemampuan suatu website dalam hal ini tampilan website untuk beradaptasi dengan ukuran layar. Lalu, apa bedanya dengan responsive web design ? Bukankah responsive juga bisa beradaptasi dengan ukuran layar ? Keduanya memang bisa menyesuaikan dengan ukuran layar, namun, dalam hal ini adaptive lebih luas cakupannya, bukan hanya bisa menyesuaikan lebar website dan elemen-elemennya, namun tampilan website bisa menjadi berbeda sama sekali antara tampilan di komputer dan di perangkat mobile. Keduanya memiliki cara pandang yang sama, yaitu menyesuaikan tampilan website dengan ukuran layar device. Tapi menurut saya sendiri lebih bagus Responsive design karena lebih fleksibel.

Apakah hadirnya smartphone dan tablet mengubah desain web secara signifikan? Bagaimana kira-kira teknologi baru akan mengubah desain web di masa depan?

Iya, Hadirnya smartphone dan tablet mengubah desain web secara signifikan karena sekarang lebih banyak pengguna smartphone / tablet itu menurut saya Tapi Kemungkinan besar sebagian orang mengakses sosial media lebih nyaman menggunakan smartphone/tablet oleh karena itu seorang web developer harus memikirkan tentang ini dengan solusi desain web yang bisa menyesuaikan device yang digunakan oleh pengguna. Jika bicara tentang teknologi baru itu pasti akan mengubah desain web di masa depan. Sebagai contohnya individu yang mempunyai keterbatasan fisik atau mental/intelektual akan lebih mudah menggunakan gadget dan mengakses web di masa depan.

Bagaimana kira-kira orang yang memiliki kekurangan fisik menggunakan website? Apakah kita perlu menyesuaikan website untuk mereka juga? Solusi apa yang dapat mengatasi keadaan tersebut?

Untuk orang dengan gangguan fisik mata (tuna netra) bisa menggunakan perintah suara untuk mengakses sebuah website tentu di website yang diakses juga di desain khusus untuk orang yang mempunyai keterbatasan fisik, Menurut saya sangat diperlukan menyesuaikan website untuk orang yang memiliki keterbatasan fisik. Solusi untuk mengatasi masalah tuna netra kita bisa mengembangkan coding untuk pengenalan suara dalam sebuah website untuk mendeteksi perintah dari pengguna.
